===============================================================================
Compile & Run Instructions
-------------------------------------------------------------------------------
Compile:
	javac HW4.java

Run:
	java HW4 option maze-file query-file

	[option] can be either "weighted" or "unweighted"
	maze-file is a .txt extension text file
	query-file is a .txt extension text file

Sample Usage:
	java HW4 unweighted RandomMaze.txt RandomQuery.txt 
	java HW4 weighted HilbertMaze.txt HilbertQuery.txt
